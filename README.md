# ProyectoFerreteriaIPOPachoclos
Este es el proyecto de IPO del grupo Pachoclos de las Prácticas de Interacción Persona-Ordenador

En index se incoporará la pantalla principal de inicio de la página web de la ferretería.
En styles.css se incluirán los estilos de la página web.
En script.js se incluirán las funcionalidades interactivas de la página web.
